◆ addPlayer()

void Game::addPlayer ( Player &  player)

Add a player to the game. Cannot be called during a game.

Used to add players to the game before the game session starts or after a game session ends. If the player array is not empty, such as after a call to this function, the game will skip player setup. Throws an exception if a game is currently running.

Parameters
playerPlayer instance to be added to the game.
